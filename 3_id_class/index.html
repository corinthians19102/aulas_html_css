ID e class

Class

Existem duas caracteristicas principais do class:

  • É possivel utilizar o mesmo class em varios elementos diferentes
  • É possivel utilizar varias classes no MESMO elemento

ID

existem duas caracteristicas principais do ID que são constantes com as caracteristicas do class

  • Não é possivel utilizar o mesmo ID em varios elementos diferentes.
  • Não é possivel utilizar VARIOS IDs no mesmo elementos.
